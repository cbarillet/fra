Fake REST API

Application de simulation d'une API REST.

Fake REST API permet de Créer un endpoint REST ou de Gérer les endpoints.

Afin de débugger vos requêtes, vous pouvez Visualiser les dernières requêtes de l'application directement sur l'interface web.

L'application est disponible sur Github.

Pour toute demande d'évolutions ou bugs, merci d'ouvrir une Issue ou de faire une Pull-request.
